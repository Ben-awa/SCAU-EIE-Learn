# 电磁场与电磁波
这门课总体来说是比较难的，知识点太多了，而想要捋清楚得至少学到麦克斯韦方程组那块，才能把知识点全部串起来  
因此，想要考前突击的朋友建议至少用一周的时间把知识点全部过一遍，

## 关于课程内容
老实说，老师挺好的，但是我听不明白，所以建议是去听网课  
然后这里附了书的电子版，可以只看电子版不用买书了(  

## 关于推荐的网课

* [电磁场与电磁波_电磁Plus](https://www.bilibili.com/video/BV11Z4y1h7oT)

期末周突击看的课程(实际上也算是系统的学习了)，**不要把这个当成突击课**  

电磁场太难了，以至于那些所谓几小时讲完的突击课讲的都不好，大多是把知识点告诉你，让你背，但是不理解的话，做题是做不出来的说

这个课就挺好的，老师语速会比较慢，喜欢慢速的可以开一倍速，我的建议是开二倍速甚至三倍速听，然后找个笔记本记笔记  
(当然不记也没关系，就是容易忘)   
然后上课的时候老师会把公式什么的都细致的推导出来，这对于对那些杂七杂八的公式的理解是很有用的。

当然缺点也是有的，更到一半断更了（悲）  
不过就我们考试的内容来说，也就平面那一块有一点点没讲吧，看学校老师ppt学习一下就好

* [电磁场与电磁波_北京交通大学_中国大学mooc](https://www.icourse163.org/course/NJTU-1002535019)

我从开学到结课期间没仔细听学校的课程，一直是听的这个，前面第一章讲的还行，但是到第二章我就有点听不下去了......
